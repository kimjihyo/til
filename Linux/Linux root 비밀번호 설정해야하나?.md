
**root 비밀번호 설정해야하는가?**


root는 Linux에서 기본적으로 추가되어있는 사용자로써 root 권한을 가지고 있다. 기본적으로 비밀번호가 설정되어있지 않다.


`sudo passwd` 커맨드를 실행해서 root의 비밀번호를 지정해줄 수 있는데 이거 꼭 해야하는건지에 대한 의문이 들었다.


우선 결론부터 정리해보자면 root 비밀번호를 설정해줄 필요도 없으며 설정하는걸 권하지 않는다고한다. 비밀번호를 설정하지 않으면 root 사용자는 비활성화되어 있으며 root로의 로그인도 불가능하다. -> 보안적으로 더 안전하다.


유저를 sudoers 그룹에 추가하여 root 권한이 필요한 명령어는 sudo를 사용하자.
